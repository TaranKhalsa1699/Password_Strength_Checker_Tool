## Features
- Instant strength scoring based on length, character variety, and patterns  
- Clear visual feedback with progress bar and color-coded results  
- Actionable suggestions to improve weak passwords  
- Checks for common sequences (123, abc, password, etc.)  
- 100% client-side – your password never leaves your browser  

## Live Demo
https://password-strength-checker-taran.streamlit.app  

## Tech Stack
- **Python** – Core logic and regex rules  
- **Streamlit** – Interactive web interface (deployed on Streamlit Community Cloud)  
- **Regex** – Pattern matching for character types and common sequences  

## How It Works
The tool evaluates passwords using the following criteria:
- Length (≥12 strongly recommended)
- Mix of uppercase and lowercase letters
- Inclusion of numbers and special characters
- Avoidance of common patterns and sequences
